# iPhone에서 확장된 화면(주소창 없음) 사용하기

## Safari 탭에서는 상단 주소창 등을 숨길 수 있나요?

**아니요. iOS Safari에서는 불가능합니다.**

- 웹 페이지가 **주소창·상단 UI·하단 바**를 숨기는 API는 **제공되지 않습니다** (Apple 정책).
- Fullscreen API는 **`<video>` 전용**이며, 문서/일반 요소에는 사용할 수 없습니다.
- 따라서 **Safari에서 탭으로 열린 상태**에서는 확장된 화면(주소창 없음)으로 만드는 방법이 **없습니다**.

---

## 확장된 화면을 쓰는 유일한 방법: **홈 화면에 추가**

**홈 화면에 추가**한 뒤, **추가된 아이콘**으로 실행하면 Safari UI 없이 앱처럼 전체 화면으로 실행됩니다.

1. Safari에서 해당 페이지 열기
2. 하단 **공유** 버튼(□↑) 탭
3. **「홈 화면에 추가」** 선택
4. 이름 확인 후 **추가**
5. 홈 화면에 생긴 **아이콘**으로 앱 실행 → 주소창·하단 바 없이 전체 화면처럼 사용 가능

이때 `manifest.json`의 `display: "standalone"` 설정으로 브라우저 UI가 숨겨진 형태로 실행됩니다.

---

## 앱에서의 안내

iPhone에서 전체 화면(⛶) 버튼을 누르면, 위 **홈 화면에 추가** 방법을 안내하는 메시지가 표시됩니다.
